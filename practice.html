Part B: True / False

  • Every link inside <ul> must be placed inside an <li>. → True
  • A page may have multiple <h1> headings to make text larger. → False
  • aria-current="page" should be used only on the active navigation link. → True
  • A form is accessible even if it has no <label> elements. → False
  • KLM predicts execution time for expert, error-free performance. → True

Part C: Short Answer

Q1: Explain the difference between a semantic element (e.g., nav) and a generic element (e.g., div).
A: Semantic elements describe their meaning and purpose for humans and machines (e.g., nav indicates navigation). Generic elements like div have no inherent meaning and are used for layout or grouping.

Q2: Why does KLM not model high-level planning and interpretation?
A: KLM is an execution-level model. It predicts performance time using low-level operators and does not model interpretation, learning, or feedback processing.
