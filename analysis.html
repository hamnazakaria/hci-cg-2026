Short-Term Memory & Chunking

Short-term memory allows humans to temporarily store a limited amount of information. Chunking groups information into meaningful units to increase memory capacity.

Syringe Problem

The syringe problem is a classic cognitive task used in HCI studies to analyze human error and design solutions that reduce mistakes.

Norman’s 7-Stage Action Cycle

Norman’s cycle describes seven stages: forming a goal, forming an intention, specifying an action, executing the action, perceiving the system state, interpreting it, and evaluating the outcome.

Gulf of Execution & Evaluation

The Gulf of Execution represents the gap between a user's goal and required actions. The Gulf of Evaluation represents the gap between system output and user understanding.

Wi-Fi Connection Example

Example of Norman’s action cycle: User forms a goal, plans, executes the action by tapping the icon, perceives feedback, interprets it, and evaluates the outcome.

Fitts’ Law

Fitts’ Law predicts the time to move to a target based on distance and size. It helps design interfaces for better usability and efficiency.
